Um app gratis, criado para facilitar as vidas dos mapas do lancer RPG!

Possui a maioria das necessidades para o funcionamento do combate(movimentação, dano, etc)!

O app pode utilizar qualquer Png ou jpg para o personagem, só se lembre de especificar o tamanho(size) no nome da foto, o default é size 1

AVISO: esse programa, assim como muitos outros, entra em conflito com o windows antivirus, para ele funcionar tente desativa-lo para esse programa
